## In a few seconds

1. Download and run one installer above.
2. Find Mochi in the Windows system tray or hidden-icons panel.
3. Left- or right-click the Mochi icon.
4. Pick a keyboard profile, mouse click pack, and volume.

Mochi supports 64-bit Windows 10 and Windows 11.

> **SmartScreen notice:** Mochi is currently unsigned, so Windows may show a warning. Confirm the installer came from this official page, choose **More info**, then **Run anyway**.

## What you get

- 12 keyboard sound profiles
- 4 mouse click packs, with separate left, right, and middle-click sounds
- Per-profile and master volume controls
- Native Windows dark/light tray menus
- Global keyboard-sound toggle: `Ctrl+Shift+M`
- Optional launch at Windows sign-in
